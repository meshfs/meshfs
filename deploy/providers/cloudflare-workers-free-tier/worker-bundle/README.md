# Worker Bundle Placeholder

This directory is for prebuilt Cloudflare Worker bundle files used by:

```bash
meshfs deploy cloudflare-workers-free-tier
```

Required files:

- `index.js`
- `index_bg.wasm`

In source repo development, these files are usually not committed.
They are generated by CI/release pipeline (or local build with `--build-worker-local`).
